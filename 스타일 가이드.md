# 스타일 가이드

> 이 보관함의 **스타일 기준 문서**. 노트를 쓸 때 여기서 문법을 복사하거나, "어떤 상황에 무엇을 쓸지"를 확인한다. Claude도 노트 작성 시 `CLAUDE.md`의 스타일 컨벤션을 통해 아래 기준을 따른다.
>
> **원칙: 목적에 맞게, 일관되게. 과용 금지 — 다양하게 쓰는 게 목표가 아니라, 의미가 있을 때만 쓰는 게 목표다.**

읽기 뷰(Reading view)로 보면 가장 정확하다. (콜아웃·수식·Mermaid는 읽기 뷰 권장)

---

## 제목 — 문서 구조
`#`~`######` (H1~H6). → **언제**: 문서를 논리적 섹션으로 나눌 때. H1은 보통 1개(제목).

# H1
## H2
### H3
#### H4

## 텍스트 강조
**굵게**, *기울임*, ***굵은 기울임***, ~~취소선~~, ==형광펜==, `인라인 코드`.
- → **굵게**: 핵심 결론·중요 단어. *남발하면 강조가 죽는다.*
- → ==하이라이트==: 나중에 다시 찾고 싶은 핵심 용어.
- → `인라인 코드`: 명령어·파일명·식별자·짧은 코드.

## 링크 — 노트 연결 (그래프의 핵심)
- 위키링크: `[[노트이름]]` → 다른 개념 노트로 연결.
- 별칭: `[[노트이름|보이는 텍스트]]` → 문장 흐름에 맞춰 표시.
- 태그: `#knowledge` `#claude-code` → 분류·그래프 군집.
- 외부: `[표시](https://...)`.

→ **언제**: 관련 개념이 나오면 무조건 링크. 고립 노트를 만들지 않는다.

## 목록
순서 없음(항목 나열):
- 항목
  - 중첩

순서 있음(단계·우선순위):
1. 첫째
2. 둘째

## 체크리스트 — Minimal 대체 체크박스
할 일과 **상태**를 아이콘으로 구분한다. → **언제**: 데일리 노트의 할 일, 액션 아이템 추적.

자주 쓰는 것:
- [ ] 할 일
- [x] 완료
- [/] 진행 중
- [!] 중요
- [?] 확인 필요
- [-] 취소
- [>] 미룸

전체 아이콘 세트:
- [<] 일정
- [*] 별표
- ["] 인용
- [l] 위치
- [b] 북마크
- [i] 정보
- [S] 금액
- [I] 아이디어(전구)
- [p] 장점
- [c] 단점
- [f] 불
- [k] 열쇠
- [w] 승리
- [u] 상승
- [d] 하락

## 콜아웃 — 강조 박스
→ **언제**: 본문에서 시각적으로 띄우고 싶은 주의·팁·정보.

> [!note] 노트 — 부가 설명

> [!tip] 팁 — 유용한 요령

> [!info] 정보 — 참고 정보

> [!warning] 경고 — 주의·함정

> [!question] 질문 — 열린 질문·확인거리

> [!danger] 위험 — 치명적 주의

## 인용구
> 외부 글 인용이나 강조 인용. → **언제**: 원문 인용, 핵심 문장 강조.

## 코드
인라인 `code`, 그리고 언어를 명시한 코드 블록:
```python
def hello():
    return "world"
```
→ **언제**: 명령어·설정·코드 스니펫. **반드시 언어를 명시**해 하이라이팅을 받는다.

## 표 — 비교·정리
| 항목 | 용도 | 비고 |
| --- | --- | --- |
| 표 | 옵션·항목 비교 | 정렬 가능 |

→ **언제**: 여러 항목을 같은 기준으로 비교/정리할 때.

## 수식 (LaTeX)
인라인 $E = mc^2$, 블록:
$$\int_a^b f(x)\,dx = F(b) - F(a)$$
→ **언제**: 수학·통계 개념 기록.

## 다이어그램 (Mermaid)
```mermaid
graph LR
  A[질문] --> B[저장] --> C[태그+링크]
```
→ **언제**: 흐름·관계·구조를 글보다 그림으로 보여줄 때.

## 각주
본문 주석을 단다.[^1]

[^1]: 각주 내용. → **언제**: 흐름을 끊지 않고 부연·출처를 덧붙일 때.

## 기타
- 구분선: `---`
- 임베드: `![[노트이름]]`, 이미지 `![[image.png]]`
- 주석(숨김): `%%렌더 안 됨%%`
- 이스케이프: `\*` `\#`

| 상황 | 스타일 |
| --- | --- |
| 핵심 결론 | **굵게** |
| 다시 찾을 용어 | ==하이라이트== |
| 주의/팁/정보 | 콜아웃 `> [!warning]` 등 |

| 항목 비교 | 표 |

| 흐름·구조 | Mermaid |
| 개념 연결 | `[[위키링크]]` + `#태그` |
